# PrivacyPolicy-Nostalgience

## Information Collection and Use

DC Enhancements built the Nostalgience app as a mobile application. This SERVICE is provided by DC Enhancements and is intended for use as is.
The app does not intentionally collect personal data beyond the technical information required for normal app delivery, network requests, and optional support communication.
- Local library entries, imported file references, save states, emulator settings, controller layouts, cheats, screenshots, cache files, and runtime logs may be stored on your device for app functionality.
- Nostasync LAN features may process local IP addresses, paired-device session data, selected game titles/file names, temporary hosted file URLs, remote input events, and frame/audio stream data between devices you pair on your own network.
- Core downloads, update checks, app resource downloads, and support requests may send normal technical request metadata to DC Enhancements infrastructure, such as IP address, user agent, requested resource, timestamps, and error information.
- Bluetooth data is used for compatible controller/gamepad connection. Bluetooth scan is not used to derive physical location.
- Microphone access is used only when an emulator core or game feature requires microphone input.
- Google Play may process purchase, entitlement, installer, and Play Integrity data for paid app access, premium unlocks, and anti-tamper verification.

## Permissions

The app may request the following Android permissions to provide its features:
- `android.permission.INTERNET` — connect to online services that provide core app functionality
- `android.permission.ACCESS_NETWORK_STATE` — detect whether a network connection is available before contacting remote services
- `android.permission.CHANGE_WIFI_MULTICAST_STATE` — support a user-facing feature that depends on this Android permission
- `android.permission.BLUETOOTH` — support a user-facing feature that depends on this Android permission
- `android.permission.BLUETOOTH_ADMIN` — support a user-facing feature that depends on this Android permission
- `android.permission.BLUETOOTH_CONNECT` — connect to nearby Bluetooth accessories selected by the user
- `android.permission.BLUETOOTH_SCAN` — support a user-facing feature that depends on this Android permission
- `android.permission.VIBRATE` — support a user-facing feature that depends on this Android permission
- `android.permission.RECORD_AUDIO` — capture microphone audio for user-facing audio features

## Third-Party Services

The app integrates the following third-party services or SDK categories:
- **Google Play Billing** — Used only for Google Play purchase or entitlement handling where paid access or premium unlocks are enabled. Google processes payments; DC Enhancements does not receive full payment card details.

## Ads and In-App Purchases

The app may offer in-app purchases. Payment processing is handled by Google Play or another configured payment provider rather than by us directly.

## Data Retention

Nostalgience keeps local library metadata, imported file references, save states, controller layouts, settings, temporary cache files, and runtime logs only as long as needed for app features, troubleshooting, or until you delete them in the app or Android app storage. Temporary Nostasync session files and hosted/remote cache files are designed to be short-lived and may be removed automatically.

## Children's Privacy

Nostalgience is a general retro-emulation utility and is not directed to children under 13. The app does not include games, ROMs, or online social features intended for children.

## Local Games, Saves, and Library Data

Nostalgience does not include or provide game files. Games and BIOS files are supplied by you from files you legally own. The app may store local library entries, file paths, platform/core selection, cover or title metadata, save states, save files, screenshots, controller profiles, cheats, emulator options, and recent-session state on your device so the emulator can work. These files remain under app/device storage unless you intentionally share, export, back up, or sync them.

## Nostasync, LAN Hosting, and Remote Compute

Nostasync features can use your local network to discover another device, host a temporary ROM/session stream, transfer inputs, or stream emulator video/audio between your own devices. This may expose local device IP addresses, session identifiers, selected game titles or file names, timing information, and temporary frame/input data to devices you pair with on the same network. DC Enhancements does not operate a public multiplayer or cloud-save service for these LAN sessions.

## Controller, Bluetooth, and Microphone Permissions

Bluetooth permissions are used to detect and connect compatible controllers and gamepads. Bluetooth scan is marked as not used for location. Microphone access is optional and used only for emulator cores that require microphone input for a game feature. Vibration is used for controller or haptic feedback where supported.

## Core Downloads and Online Requests

When you use features that download emulator cores, compatibility assets, update information, or app resources, Nostalgience may contact DC Enhancements servers such as dcenhancements.com. Server logs may contain normal technical request metadata such as IP address, user agent, requested file, timestamps, and error status for security, delivery, abuse prevention, and troubleshooting.

## Google Play Billing and Play Integrity

If Nostalgience is distributed as a paid app or uses premium unlocks, purchase handling is performed through Google Play. Production builds may use Google Play Billing and Play Integrity to verify legitimate installation and protect the app against tampering or unauthorized distribution. DC Enhancements does not receive your full payment card details; Google handles payment processing.

## Legal Game Ownership

Nostalgience is an emulator front end/runtime utility. No commercial games, copyrighted BIOS files, or ROM downloads are included. You are responsible for using only game and BIOS files that you are legally allowed to use in your region.
